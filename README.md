# ESP32 Custom PCB

This is a custom-designed ESP32 development board created using EasyEDA. It includes:
- Power input management
- USB-to-Serial interface
- ESP32-WROOM-32E module
- Reset & Boot buttons
- LED indicators
- Gerber and BOM files for fabrication
